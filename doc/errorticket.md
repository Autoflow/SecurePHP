## Example usage:

```php

try {

    // start Autoflow\SecurePHP
    $mysecure = AUTOFLOW\SECUREPHP\BOOTSTRAP::getInstance(true, false);

    // set to strict mode aborting on E_NOTICES
    $mysecure->loose(false);

    // try to access a unset variable
    echo $x;

    }

```
