## Example usage:

```php

try
    {

    // start Autoflow\SecurePHP
    $mysecure = AUTOFLOW\SECUREPHP\BOOTSTRAP::getInstance(true, false);

    // display reports
    $mysecure->mute(false)

    // create and raise error ticket
    $ticket = new Warning('Database update failed today', 'Retry in 30 minutes');
    $ticket->raise();


```
